# Architecture

## High-Level Design

Venue WiFi
 -> NanoStation loco M5
 -> Beryl AX Router
 -> EAP225 Access Point
 -> Android Cameras
 -> OBS Studio

## Why This Design

Commercial wireless HDMI solutions scale poorly and become expensive when multiple cameras are required.

Using smartphones and IP networking reduces cost while leveraging existing networking knowledge and tools.

## Core Technologies

- DroidCam OBS
- OBS Studio
- WiFi 5 (5 GHz)
- DHCP
- NAT
- Wireless Bridging

# Troubleshooting

## Beryl AX WAN Issues

Symptoms:
- Internet randomly unavailable
- Cameras remain connected

Root Cause:
WAN connection status checking incorrectly detects outages.

Fix:
Disable WAN connection status checking.

---

## MAC Address Issues

Symptoms:
- WAN appears connected
- No internet access

Fix:
Clone MacBook MAC address on WAN interface.

---

## Camera Dropouts

Potential Causes:
- AP placement
- RF congestion
- Distance

Mitigation:
- Raise AP
- Reduce obstacles
- Use 5 GHz

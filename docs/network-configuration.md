# Network Configuration

## NanoStation

Purpose:
- Connect to venue WiFi
- Deliver WAN connectivity to the streaming network

Recommended:
- Bridge mode
- Static management IP

## Beryl AX

Responsibilities:
- NAT
- DHCP
- Gateway

Suggested IP Plan

192.168.8.1 Router
192.168.8.2 AP
192.168.8.10 OBS

Reserve camera addresses.

## EAP225

Settings:
- 5 GHz only
- 40 MHz width
- Medium power
- Fast roaming disabled
- Mesh disabled

## AP Placement

Mount 8-10 feet above floor level.

Benefits:
- Better line-of-sight
- Reduced player interference
- Better 5 GHz performance

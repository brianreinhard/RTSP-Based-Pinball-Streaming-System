# Network Configuration

## NanoStation

Purpose:
- Connect to venue WiFi
- Deliver WAN connectivity to the streaming network

Recommended:
- Bridge mode
- Static management IP

## Beryl AX

Responsibilities:
- NAT
- DHCP
- Gateway

Suggested static IP assignments

192.168.50.2 - WiFi bridge 
192.168.50.1 - Router  
192.168.50.3 - AP  
192.168.50.10 - OBS  
192.168.50.11 - Camera 1  
192.168.50.12 - Camera 2  
192.168.50.13 - Camera 3  


## EAP225

Settings:
- 5 GHz only
- 40 MHz width
- Medium power
- Fast roaming disabled
- Mesh disabled

## AP Placement

Mount 8-10 feet above floor level.

Benefits:
- Better line-of-sight
- Reduced player interference
- Better 5 GHz performance
